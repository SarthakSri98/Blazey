How to Use?

1. Click on the '+' button on the right and give your genre name. For ex: Movies.

2. On the next page there will be topics. Here you can give topics related to the genre above. For eg. Action Movies,Horror Movies or the movie name etc.

3. On the topics part you have two options:
(i) If you select no then a few more input fields will be provided to you namely:
(a) link : Any sort of valid link
(b) linkCaption : A caption for that link
The link and linkcaption will be presented in pairs and you can as many of the links as you want.
(c) Description : The Description about that topic.
If you select this then after clicking the card, you will see a modal with the data you had given earlier. For ex. Titanic movie , it's link with website name in linkcaption(say torrent) , and then a bit of description about the movie.

(ii) The second option is that you can opt for further subtopics by selecting yes to the question "Will there be any subtopic?"
If you select this then after clicking the card, you will be routed to a new page named "Subtopics inside `the topics you had given earlier`". For ex. If you are in the action movie topic then you can give avengers, justic league as a sub-topic.

4. On the subtopics section you can have the same results as 3.(i) has.

So there are two possible flows :
(i) Genre -> Topic -> end.
(ii) Genre -> Topic -> Sub-Topic -> end
